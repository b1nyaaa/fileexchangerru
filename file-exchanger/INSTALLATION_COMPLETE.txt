                       💡 ПОЛЕЗНЫЕ КОМАНДЫ WINDOWS
═══════════════════════════════════════════════════════════════════════════

Узнать IP адрес:
  1. Запустите файл: get_ip.bat
  2. ИЛИ откройте PowerShell и выполните: ipconfig

Проверить сеть:
  1. Запустите файл: network_test.bat
  2. Получите информацию о сети и портах

Остановить сервер:
  • Нажмите Ctrl+C в окне PowerShell где запущен сервер

Запустить сервер вручную (если start_server.bat не работает):
  1. Откройте PowerShell
  2. Перейдите в папку:
     cd C:\Users\<ваше_имя>\Desktop\file-exchanger\server
  3. Запустите:
     python server.py

═══════════════════════════════════════════════════════════════════════════
                         ⚡ БЫСТРЫЕ РЕШЕНИЯ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════

ПРОБЛЕМА: "Ошибка подключения"
РЕШЕНИЕ:
  1. Проверьте IP адрес (запустите get_ip.bat)
  2. Убедитесь что оба компьютера в ОДНОЙ сети
  3. Убедитесь что сервер запущен (окно PowerShell открыто)
  4. Проверьте брандмауэр Windows

ПРОБЛЕМА: Брандмауэр блокирует
РЕШЕНИЕ:
  1. Откройте Windows Параметры
  2. Перейдите: Конфиденциальность → Брандмауэр Windows
  3. Нажмите "Разрешить приложению через брандмауэр"
  4. Добавьте Python или порт 8888

ПРОБЛЕМА: Медленная загрузка файлов
РЕШЕНИЕ:
  1. Это проблема сети, не приложения
  2. Попробуйте проводное подключение (LAN кабель)
  3. Закройте другие приложения что используют интернет
  4. Используйте более быструю сеть

ПРОБЛЕМА: start_server.bat не открывается
РЕШЕНИЕ:
  1. Откройте PowerShell
  2. Перейдите в папку: cd file-exchanger/server
  3. Запустите: python server.py

   C:\Users\<ваше_имя>\Desktop\file-exchanger\server\shared_files
   Здесь хранятся все загруженные файлы

💡 Обновите список файлов:
   Нажмите F5 в браузере

💡 Очистите кэш браузера:
   Если интерфейс не обновляется, нажмите Ctrl+Shift+Delete
