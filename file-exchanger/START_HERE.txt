Q: Нужно ли что-то устанавливать на клиента?
A: НЕТ! Только браузер.

Q: Нужен ли интернет?
A: НЕТ! Работает в локальной сети.

Q: Это безопасно?
A: ДА для локальной сети. Для интернета нужна VPN.

Q: Какие файлы можно отправлять?
A: ЛЮБЫЕ! Размер ограничен памятью.

Q: Сколько людей может подключиться?
A: НЕОГРАНИЧЕНО! 

Q: Где хранятся загруженные файлы?
A: В папке: server/shared_files/

❌ Не подключаюсь
   ✓ Проверьте IP адрес (run get_ip.bat)
   ✓ Оба ПК должны быть в одной сети
   ✓ Убедитесь что сервер запущен

❌ Брандмауэр блокирует
   ✓ Откройте брандмауэр для портала 8888
   ✓ Разрешите Python доступ

❌ Медленная загрузка
   ✓ Это проблема сети, не приложения
   ✓ Используйте проводное подключение (LAN)

Полное решение проблем в: GUIDE.md

═══════════════════════════════════════════════════════════════
         🌟 СОВЕТЫ И ТРЮКИ
═══════════════════════════════════════════════════════════════

💡 Добавьте start_server.bat в автозагрузку
   (папка: C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup)

💡 Используйте проводное подключение (LAN) для скорости

💡 Откройте server/shared_files/ чтобы управлять файлами

💡 Обновите список файлов: F5 в браузере

💡 Измените IP адрес в адресной строке для другого ПК
