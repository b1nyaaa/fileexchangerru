📡 REST API
═══════════════════════════════════════════════════════════════

GET /
  └─ Возвращает: HTML интерфейс (index.html)

GET /api/files
  └─ Возвращает: JSON список всех файлов
  └─ Формат: [{"name": "file.txt", "size": 1024, ...}, ...]

GET /api/download/<filename>
  └─ Возвращает: Бинарные данные файла
  └─ Заголовок: Content-Disposition: attachment

POST /api/upload
  ├─ Параметр: X-Filename (имя файла в заголовке)
  ├─ Тело: Бинарные данные файла
  └─ Возвращает: JSON {status: "success", filename: "..."}

DELETE /api/delete/<filename>
  └─ Возвращает: JSON {status: "success"}

GET /api/server-info
  └─ Возвращает: JSON {local_ip: "...", port: 8888, status: "online"}
